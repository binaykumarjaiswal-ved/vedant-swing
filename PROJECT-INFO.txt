VEDANT SWING — MORNING-ONLY RECOMMEND SYSTEM
============================================

PRODUCT: Vedant Swing (India Nifty 500 swing recommendations)
REPO: https://github.com/binaykumarjaiswal-ved/vedant-swing
LIVE APP: https://vedant-swing-web.onrender.com
DATA: Yahoo Finance (yfinance) + NSE quotes
BROKER: NONE — no auto trade, no order placement
GOALS: See GOALS.md
MODE: Morning research only (evening scan removed)

FEATURES:
  [x] Nifty 500 universe
  [x] Morning research + BUY recommendations
  [x] Market regime filter
  [x] ATR risk: target/stop + risk-based position size
  [x] Hard stop + max 1 optional average
  [x] Confidence-gated recommendations
  [x] SQLite history + model audit + backtest
  [x] Auto paper trade (virtual only)
  [x] Charts, watchlist, alerts, journal
  [x] Mobile web dashboard
  [x] GitHub Actions + Render (morning)

REMOVED (do not re-add without product decision):
  - evening_scan.py / evening GitHub workflow
  - swing_strategies evening packaging
  - evening email/Telegram summaries
  - evening UI card / APIs

QUICK COMMANDS:
  RUN_RECOMMEND.bat   — today's BUY recommendations
  RUN_BACKTEST.bat    — historical sample backtest
  RUN_MODEL_AUDIT.bat — score past predictions

DAILY ROUTINE:
  8:30–10:30 AM  Morning research + BUY recommendations
  Market hours   Manual broker trade if you agree; check stop/target

PHONE: https://vedant-swing-web.onrender.com
